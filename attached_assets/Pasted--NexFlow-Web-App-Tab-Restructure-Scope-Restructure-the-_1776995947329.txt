# NexFlow — Web App Tab Restructure

**Scope:** Restructure the current NexFlow web app navigation. Consolidate scattered tabs, fix broken features, redefine weak ones.
**Language:** English only (for this phase)
**Output:** Developer-ready spec for Replit build.

---

## 1. Target Navigation — 6 Primary Tabs

Everything in the app collapses into these six tabs. Everything else is a sub-tab.

| # | Primary Tab | Sub-Tabs |
|---|---|---|
| 1 | **Home** | Daily Briefing · Performance · To-Do · Daily Insights · AI Assistant *(persistent panel)* |
| 2 | **Leads** | Pipeline & Deals · Lists · Research · Forgotten Leads |
| 3 | **Call Center** | Dashboard · Calls & Transcripts · AI Agent · Knowledge Base · Messages *(WhatsApp + Email)* |
| 4 | **Data Hub** | Segments · Enrichment · AI Workforce · Signals |
| 5 | **Marketing** | Campaigns · Templates · Audiences |
| 6 | **Insights** | Dashboards · Reports · Team Performance · Predictive |

Settings lives as a gear icon / user menu — not a primary tab.

### What moved and why

| Current location | New location | Reason |
|---|---|---|
| Separate "Deals" + "Pipeline" tabs | Merged under Leads → Pipeline & Deals | Same object, two views |
| Separate "List View" + "Static List" tabs | Merged under Leads → Lists | Same object, one view with a toggle |
| Top-level "Lead Intelligence" | Sub-tab under Leads → Research | Rename clarifies purpose |
| Top-level "Activities" | Removed — lives inside each Lead profile | Was noise, not a destination |
| Top-level "Signals" | Sub-tab under Data Hub | Back-office feed, not a primary workflow |
| Top-level "Segments" | Sub-tab under Data Hub | Feeder for campaigns/prospecting, not a daily destination |
| Top-level "AI Workforce" | Sub-tab under Data Hub | Back-office control panel |
| "Campaigns" inside Call Center | New top-level Marketing | Outreach ≠ calling; needs its own home |
| "Call Scripts" as standalone | Sub-tab under Call Center → Knowledge Base | Lives with objections and playbooks |
| WhatsApp + Email as separate tabs | Merged under Call Center → Messages | One unified conversation view |

## 2. Home — Consolidated Briefing

The Home tab becomes the **single interactive landing page** a user sees each morning. No separate "briefing" vs "performance" vs "to-do" tabs.

### Layout — 5 sections on one screen

**A. Daily Briefing card (top)**
- Date and a plain-language summary of the day
- Pulls from: today's calendar, overdue tasks, hot leads, pipeline changes overnight
- Example: *"3 hot leads need follow-up, 2 callbacks due by noon, 1 deal stalled over 14 days."*

**B. Performance strip**
- KPI tiles: Calls today · Meetings booked · Pipeline value · Win rate (7d / 30d / YTD)
- Sparklines on each tile
- Click any tile → drills into Insights → Dashboards

**C. To-Do list (interactive)**
- Today's actions from: HubSpot tasks, Forgotten Leads engine, Signals
- Each row: lead name, action type, urgency, deep link to lead
- Inline actions: ✓ Done · ⏭ Snooze · 📞 Call · 💬 Open

**D. Daily Insights feed**
- Auto-generated overnight: market news hitting portfolio companies, new wealth signals detected, regulatory changes, deal risk alerts
- Dismissible cards, each with a "Take action" CTA

**E. AI Assistant (persistent right rail, all screens)**
- Chat panel available from Home and from every other tab
- Has context of whatever screen the user is on
- Suggested prompts rotate by screen: *"Summarize my day"*, *"Which leads am I ignoring?"*, *"Draft follow-ups for today's callbacks"*
- Powered by Claude API (`claude-sonnet-4-6`)

### Acceptance criteria
- All 5 sections load in one scroll (no tab-switching inside Home)
- AI Assistant reads current page context
- To-Do items are clickable and write back to HubSpot
- Performance tiles refresh at least every 15 minutes

## 4. Forgotten Leads — Redefined

### 4.1 New definition
A lead is "Forgotten" when **BOTH** conditions are true:

1. **No contact for ≥ 90 days** (no call, email, WhatsApp, or meeting logged), AND
2. **At least one signal** from this list:

| Signal | Trigger |
|---|---|
| 💰 Wealth event | Liquidity event, IPO exit, property transaction, dividend declaration linked to contact or their company |
| 💼 Job change | New role detected via LinkedIn, press, or regulatory disclosure |
| 🏛 Inheritance | Obituary or estate event linked to family |
| 📱 Digital activity | Opened recent email, visited website, engaged on LinkedIn, podcast appearance |
| 🤝 Broken promise | CRM note contains *"follow up"*, *"callback"*, *"send proposal"*, *"revisit"* with no subsequent activity |
| ⏰ Unactioned task | Task due > 30 days ago, never closed |

### 4.2 UI — replace current harsh color palette
Signals display as **soft chips** on each lead row. Current colors are too saturated. Replace with:

| Signal | New color |
|---|---|
| Wealth event | Soft lavender |
| Job change / digital activity | Muted sky blue |
| Inheritance / broken promise | Warm sand |
| Unactioned task | Muted coral *(not red)* |

Each chip has a tooltip: *"Why is this here?"* → explains the exact trigger.

### 4.3 Bulk actions on the list
- Assign to me
- Add to sequence
- Dismiss for 30 days
- Export

### 4.4 Query logic
```
forgottenLeads = contacts WHERE
  last_activity_date < NOW() - 90 days
  AND (
    signals.wealth_event = true
    OR signals.job_change = true
    OR signals.inheritance = true
    OR signals.digital_engagement_last_30d = true
    OR notes MATCHES /follow up|callback|revisit/i
    OR tasks.due_date < NOW() - 30 days AND status != 'completed'
  )
ORDER BY signal_recency DESC
```

## 5. Call Center — Full Redesign

### 5.1 Dashboard *(new landing — replaces current weak entry)*
Analytics-first landing page.

**Widgets:**
- **Today's calls** — count, duration, connect rate
- **Activity heatmap** — calls by hour × day, last 30 days
- **Trend chart** — calls per week, with YoY toggle
- **Leaderboard** — top performers (team-level)
- **AI Insights panel** — auto-generated, e.g. *"Connect rate drops 40% after 4pm — shift calls earlier"* or *"Leads contacted within 2 min of a signal convert 3x"*
- **Predictive analytics**:
  - Closing probability per open deal (model-scored)
  - Success analysis — which call patterns correlate with wins

**Controls:**
- Date range selector: Today · 7d · 30d · QTD · YTD · Custom · YoY comparison
- "Generate Report" button → exports PDF/XLSX of current view

### 5.2 Calls & Transcripts *(redesigned)*

**During a live call:**

Three-column layout:
- **Left rail — Lead context**: who they are, last conversation summary, research dossier highlights
- **Center — Live transcript**: scrolling, speaker-labeled, bilingual support (Arabic/English/Gulf/Levantine/Egyptian dialects)
- **Right rail — Live coaching panel (human-driven, NEW)**:
  - Senior BD can type notes in real time that appear on the rep's screen
  - Suggested scripts auto-surface based on keywords detected in transcript
  - **Objection handler**: when an objection is detected (*"too expensive"*, *"already have a provider"*, *"send me an email"*), top 3 pre-approved responses pop up instantly

**Post-call:**
- Full transcript with speaker labels + timestamps
- AI-generated: summary · key quotes · action items · sentiment score
- "Add to CRM" button → writes note back to lead record

### 5.3 AI Agent
**Same interface** as Calls & Transcripts for humans — so reps supervise AI calls the same way they review their own.

Additional toggle row at the top:
- 🎙 **Voice selector** — dropdown of available voices (nova, onyx, shimmer, echo, fable, alloy)
- 🗣 **Dialect selector** — Gulf · Levantine · Egyptian · MSA · English
- 🎛 **Persona selector** — pick from saved personas

### 5.4 Knowledge Base *(consolidates scattered content)*
One home for everything that supports calls.

- **Scripts** — by scenario: cold call · follow-up · objection · closing
- **Objection library** — each objection maps to 3 approved responses, tagged by client type
- **Playbooks** — structured sequences (e.g. *"HNW family office first touch"*, *"Post-IPO follow-up"*)
- **Company insights** — firm pitch, differentiators, compliance talking points

Searchable, filterable, versioned. Each item links from the live coaching panel in 5.2.

### 5.5 Messages *(WhatsApp + Email merged)*
One unified timeline per lead, channel icon per message.

- **Generated from the lead itself** — the send buttons live on the lead profile
- This tab is the **outbox / history / inbox view**
- Quick-send templates: HNW intro, meeting confirm, document request
- Filter by: channel, status (sent/delivered/read/replied), owner

### 5.6 Removed from Call Center
- ❌ **Campaigns** → moved to Marketing (Section 7)

## 8. Insights *(expanded)*

### 8.1 Dashboards
- Prebuilt: Pipeline Health · Conversion Funnel · Activity Quality · Segment Performance
- Every dashboard is **editable** — add/remove widgets, adjust filters
- Save custom dashboards per user

### 8.2 Reports
- Generate any report from any dashboard
- Export: PDF · XLSX · PPTX
- Schedule: daily / weekly / monthly email delivery

### 8.3 Team Performance
Per-rep scorecards with **three required filters**:
- **YTD** (current year)
- **Since inception** (all-time from firm founding)
- **Custom range**

Metrics: calls · meetings booked · proposals sent · deals won · AUM added · revenue generated.

The dashboard itself is **adjustable** — users can modify which KPIs show and add analysis widgets (e.g. correlation between call volume and close rate). YoY comparison toggle on every metric.

### 8.4 Predictive
- Deal closing probability (model-scored per open deal)
- Success pattern analysis — which lead attributes + activity patterns correlate with wins
- Churn risk scoring for existing clients
